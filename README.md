# OpenCode Configurations

This repository contains my personal configurations for OpenCode, including my custom agents.

## Agents

| Agent | Description |
| :--- | :--- |
| **Agent Editor** | Manages and edits OpenCode agents across global and project locations. |
| **Mira** | A strict Senior Pair Programmer focused on "teaching by doing" through direct tasks and technical constraints. |
| **Quest** | Answers questions using built-in knowledge or external resources without local file access. |

## Commands

| Command | Description |
| :--- | :--- |
| `/quest` | Search for answers without local file access. |

For more information on agents or OpenCode, visit the [official documentation](https://opencode.ai/docs/agents/).
